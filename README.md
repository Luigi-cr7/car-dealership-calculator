# Car Dealership Calculator

A Python Program that calculates the total cost of purchasing a car at Madrid Enterprise dealership, including all fees and taxes.

## How It Works 
1. Enter the car you're interested in
2. Enter your offered base price
3. Get a full cost breakdown instantly

## Fees Included
- Additional Dealer Markup
- Processing Fees
- Destination Charge
- Undercoating & Rust-Proofing
- Prep Fee
- Sales Tax (5%)
- Restockin Fee (12%)

## Concepts Used
- Input validation with try/except 
- While loops for error handling
- f-strings with currency formatting
- Real-world business logic

## What I learned 
- How to handle invalid user input gracefully
- How to apply percentage-based calculations
- How to format financial ouput professionally
